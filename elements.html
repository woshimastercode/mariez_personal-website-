Projects

Below are several projects that reflect my interests in statistics, data analysis, and art education. Each project focuses on a different problem and explores how analytical thinking and creative approaches can be applied to understand real-world questions.

Credit Card Fraud Detection

This project analyzed financial transaction data to better understand patterns related to fraudulent activity. I first conducted exploratory data analysis to examine how factors such as transaction amount, location, and merchant category relate to fraud risk.

After understanding the dataset, I developed classification models including logistic regression and random forest. These models were used to estimate the probability that a transaction was fraudulent. The project helped me learn how predictive models can identify suspicious patterns and how model evaluation helps determine whether a model performs reliably.

Skin Cancer Prediction

This project focused on predicting skin cancer outcomes using machine learning models. The dataset contained medical information related to skin lesions, and the goal was to classify whether a case was malignant or benign.

I implemented several models including logistic regression, random forest, and K-nearest neighbors. By applying cross-validation, I compared the predictive performance of each model. The project allowed me to understand how different algorithms behave when solving classification problems.

Hyperparameter Optimization Study

This research project explored how different hyperparameter tuning strategies influence machine learning model performance. In particular, the project compared grid search, random search, and Latin hypercube sampling.

Using a gradient boosting model, I generated multiple parameter configurations and evaluated their predictive performance. The goal was to understand how different search strategies balance computational cost with model performance.

Gender and Gaming Culture Analysis

This project examined how audiences interact with gaming influencers on social media and whether gender influences audience responses.

Using social media posts and user interactions, I analyzed patterns in audience engagement and discussion topics. The analysis explored how gender representation may influence the way audiences respond to gaming content creators.

Art Education Teaching Activity

This project involved designing a classroom activity that uses visual art to support reflection and creative expression. Students were encouraged to communicate ideas through drawing and visual storytelling.

The goal of the activity was to explore how artistic practice can support critical thinking and emotional expression in learning environments. This experience helped me understand how creative teaching strategies can encourage participation and engagement.
